# Pem Colak — kartvizit paylaşım linkleri (UTM)

Kartı **nerede paylaşıyorsan o kanalın linkini kullan.** Etiketsiz (çıplak)
https://www.pemcolak.xyz linkiyle gelenler GA'da "Direct" görünür — kaynağı bilemezsin.

GA'da görmek için: **Reports → Acquisition → Traffic acquisition** → Source / Medium kırılımı.
Kampanya hepsinde `card` → tek kampanyada toplanır, kanal bazında ayrışır.

| Kanal | Link |
|---|---|
| LinkedIn (profil / banner) | https://www.pemcolak.xyz/?utm_source=linkedin&utm_medium=profile&utm_campaign=card |
| LinkedIn (gönderi) | https://www.pemcolak.xyz/?utm_source=linkedin&utm_medium=post&utm_campaign=card |
| Instagram (bio) | https://www.pemcolak.xyz/?utm_source=instagram&utm_medium=bio&utm_campaign=card |
| WhatsApp | https://www.pemcolak.xyz/?utm_source=whatsapp&utm_medium=chat&utm_campaign=card |
| E-posta imzası | https://www.pemcolak.xyz/?utm_source=email&utm_medium=signature&utm_campaign=card |
| Basılı kart / QR kod | https://www.pemcolak.xyz/?utm_source=qr&utm_medium=print&utm_campaign=card |
| Yüz yüze (toplantı) | https://www.pemcolak.xyz/?utm_source=inperson&utm_medium=meeting&utm_campaign=card |

## Notlar
- Yeni kanal eklemek istersen: `utm_source` = nereden (linkedin, behance…),
  `utm_medium` = nasıl (post, bio, dm…), `utm_campaign` = `card` sabit kalsın.
- Linki kısaltmak istersen bit.ly / kısa link kullanabilirsin; UTM parametreleri
  yönlendirme sonunda korunur.
- QR kodu basılı kart için: yukarıdaki `qr` linkini bir QR üretecine ver → baskıdan
  gelen taramaları ayrı sayarsın.
